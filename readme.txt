GEPO - Gestión de Proyectos Online

Pasos para el levantamiento del proyecto:

1) Abrir una consola en la carpeta raíz del proyecto y ejecutar el comando `npm start` para levantar el servidor.

2) Una vez levantado se conectará automaticamente a la base de datos la cual está montada en un servidor web llamado `mLab MongoDB`.

3) Ahora ya podremos ir al localhost:3333/ para acceder al Langing Page y comenzar a navegar por la página.

Acceder a la aplicación web:

Llendo al link https://gepo-hn.herokuapp.com/ podremos acceder a la aplicación web montada en Heroku.
